# pixcat

A python module to display an image in the kitty terminal

This script is inspired from unmaintained [mirukana/pixcat](https://github.com/mirukana/pixcat/tree/69e35c0cbe639bee16baf31c0b7954dbf45621a1) and [the kitty image protocole specification](https://github.com/kovidgoyal/kitty/blob/master/docs/graphics-protocol.rst).
